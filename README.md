# HerCycle 🌸

HerCycle is your intelligent companion for menstrual health tracking and wellness.

## ✨ Features

- 📅 Smart Calendar
  - Track PMS, period, and ovulation days
- 🤖 AI Health Assistant
  - Ask questions about your health
- 🔔 Custom Reminders
  - Never miss important actions
- 🔐 Secure & Private
  - Your data is protected

## 🛠 Tech Stack

- 📱 iOS: SwiftUI & UIKit
- 🏗 Architecture: MVVM
- 🔥 Firebase
  - Authentication
  - Realtime Database
  - Firestore
- 🌐 Custom Network Package
- 🧠 OpenAI Integration

## 📱 App Structure

- **Models**: Core data structures
- **Views**: User interface components
- **ViewModels**: Business logic and state management
- **Services**: 
  - Firebase integration
  - Network operations
  - AI assistant functionality

## 🔐 Security

- Secure authentication via Firebase
- Encrypted data storage and transmission

## 🚀 Performance

- Efficient real-time data syncing
- Optimized for iOS 15.0+
